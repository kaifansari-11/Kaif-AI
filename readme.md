# Kaif AI

A modern AI chat application powered by Puter.js with support for multiple AI models, conversation memory, file uploads, markdown rendering, syntax highlighting, and image attachments.

## Live Demo

https://kaif-ai.netlify.app

---

## Tech Stack

| Layer               | Technology                         |
| ------------------- | ---------------------------------- |
| Frontend            | HTML, CSS, JavaScript              |
| AI Provider         | Puter.js                           |
| Models              | Claude Sonnet 4.6, Claude Opus 4.8 |
| Storage             | Browser LocalStorage               |
| Markdown            | Marked.js                          |
| Syntax Highlighting | Highlight.js                       |
| PDF Parsing         | PDF.js                             |
| Excel Parsing       | SheetJS (XLSX)                     |

---

## Features

### AI Chat

* Multi-model support
* Claude Sonnet 4.6
* Claude Opus 4.8
* Persistent conversation history

### Memory System

* Chat history stored in LocalStorage
* Previous conversations restored automatically
* Context-aware conversations

### File Uploads

* TXT files
* Markdown files
* HTML, CSS, JavaScript
* JSON
* Python
* Java
* C++
* PDF documents
* Excel spreadsheets (.xlsx, .xls)

### Image Support

* Upload images
* Paste screenshots directly (Ctrl + V)
* Image preview before sending
* Vision-ready architecture

### Developer Features

* Markdown rendering
* Syntax highlighted code blocks
* Clear chat button
* Model switching
* Dark theme UI

## Local Setup

### Prerequisites

* Modern Browser
* Internet Connection
* Puter Account

### Run

Simply open:

```html
index.html
```

No backend required.
